Homelab

Production-grade infrastructure at home

My homelab isn't a school project — it's a real multi-node environment I've been building and maintaining for years. It runs 24/7 and serves as my primary testbed for learning systems administration in practice.

🖥️

Compute

Two Proxmox nodes (Ryzen 7 6800H) plus a rackmount server (Ryzen 9 3900X). VMs and LXCs across all nodes, managed via Proxmox VE.

🌐

Network

UniFi stack with 9 VLANs — Management, Storage, Services, Trusted, IoT, Guest, and more. IDS/IPS via UCG Fiber, Technitium DNS.

🗄️

Storage

Primary NAS on TrueNAS SCALE with RAIDZ2 (5×10TB). Backup NAS with daily ZFS replication and 30-day retention.

⚡

Power & Reliability

APC UPS with NUT running a progressive shutdown strategy — TrueNAS at 5 min, compute nodes at 10–15 min, coordinated via Ansible.

🔧

Automation

Ansible manages all 20 hosts with SSH key auth, SSH hardening, and NUT config across the fleet. All changes tracked in Git.

📊

Self-Hosted Services

Nextcloud, Vaultwarden, Jellyfin, Immich, full *arr stack, Uptime Kuma with Discord alerts, BookStack documentation.
